# <strong>Installation</strong>

1- Clone this repo


2- Run the command

```bash
yarn && yarn dev
```

or

```bash
npm i && npm run dev
```

<br/>

# Tests

1- Just run the command

```bash
yarn test
```

or

```bash
npm run test
```
